============================================
 ITSプロジェクト向けSound Map描画プログラム
============================================

音声ファイルの左右音声の類似性をGeneralized Cross-Correlationを使って判断するプログラム。

Required Libraries
==================

* wave
* numpy

Usage
=====

* ``python main.py <wavefile>`` のように音声のwavファイルを指定して実行する。
